CanApp Cards

These apps cards should be present on all CanApps to promote other CanApps.
These cards are generated using cancards.js.


To create these cards, simply dollow these steps:

  • In Angular, you copy and paste these codes in your index.html file. 
    <!-- Put this at the top of the file -->
    <link rel="stylesheet" href="https://canya.io/assets/cancards.css">
    <!-- Put this at the bottom of the file -->
    <script src="https://canya.io/assets/cancards.js"<>/script>
  • Copy the following code in the page where you want to generate the cards in : 
    <div class="canAppDiv">
         <div class="canyaCard"></div>
    </div>
  • For non-angular projects, you can now generate the cards by simply calling the function you wanted. e.g. 
    <script>createCards()</script>
  • For angular projects, you'll have to call the function from the component's .ts file. First, declare the function you want to use. e.g. 
    declare var createCardsRandomly:any
    And call the function inside the ngOnInit function, e.g.
    ngOnInit() {
    createCardsRandomly(8,"");
    }

Learn more about it on its Github page
