Bancor Widget

This widget must be present in all CanApps.

To put this widget, follow these steps :

  • Add this code into your HTML; 
    <div id="bancor-wc"></div>
  • Insert this code at the end of your HTML file: 
    <script src="https://widget-convert.bancor.network/v1"></script>
    
<script>
BancorConvertWidget.init({
"type": "1",
"baseCurrencyId": "5a6f61ece3de16000123763a",
"pairCurrencyId": "5937d635231e97001f744267",
"primaryColor": "#00BFFF",
"primaryColorHover": "#55DAFB"
});
</script>
  • Some notes on the variables, and what you can do with it: 
    "primaryColor":  - change this if you would like to modify the main widget color from black
"primaryColorHover":  - change this if you would like to modify the "mouse over" effect of the buttons
"hideVolume": true - add this line if you want to hide the 24h trade volume)    
"unlockBaseCurrency": true - add this line if you want to enable your visitors to convert any token to any token on the Bancor network (default is limited to your token and token relay only)
